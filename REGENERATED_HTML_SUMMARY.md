#### 高反预防指南（重要！）

**提前准备**：
- 📅 提前3天给孩子吃红景天口服液（儿童装）
- 💤 前一晚早睡，保证充足睡眠
- 🍳 早餐吃饱，带巧克力、糖果补充能量

**现场注意**：
- 👀 随时观察孩子状态（脸色、精神）
- 🚫 不要让孩子剧烈奔跑
- 💧 多喝热水，少量多次
- 🫁 头晕、恶心立即吸氧

**高反症状判断**：
- 轻度：头晕、头痛 → 吸氧、休息
- 中度：恶心、呕吐 → 吸氧、立即下山
- 重度：嘴唇发紫、呼吸困难 → 紧急下山+就医

#### 必备物品清单

**必须带**：
- [ ] 氧气瓶 × 4（必备！宁可不用，不可没有）
- [ ] 羽绒服或厚外套（山上5-10°C）
- [ ] 手套（孩子必备，可以玩雪）
- [ ] 帽子（防晒+保暖）
- [ ] 墨镜（2大2小都要！紫外线超强）
- [ ] 防晒霜 SPF50+
- [ ] 零食、巧克力、糖果
- [ ] 保温杯+热水
- [ ] 儿童常用药（感冒药、止泻药、创可贴）

**建议带**：
- [ ] 润唇膏（高原很干燥）
- [ ] 暖宝宝贴
- [ ] 备用衣物（孩子出汗可换）
- [ ] 防滑手套（有雪可以玩）
